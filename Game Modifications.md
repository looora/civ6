# Turn off unit auto cycle?
Go to `Documents/My Games/Sid Meyer's Civilization VI`, open the `UserOptions.txt` and change the value to 0.  
```ini
;Does the selection auto cycle to the next available unit? (0 = no, 1 = yes)  
AutoUnitCycle 0
```
